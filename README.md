# john-henry-hypercard

Web recreation of the They Might Be Giants "John Henry" HyperCard stack from 1994.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
